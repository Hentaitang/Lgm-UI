## 开始使用
1. 安装
    
2. 添加 CSS 样式
    使用本框架前，请在 css 中开启 border-box
    ```$xslt
    *, *::after, *::before{
        box-sizing: border-box;
    }
    ```
    IE8 及以上浏览器都支持此样式
